🌦️ Weather Finder (Pincode Based)

A simple web application that fetches **weather details using pincode**.
Built using **HTML, CSS, and JavaScript** with OpenWeatherMap API.


🚀 Features

* Enter any **Indian pincode**
* Fetch real-time weather data
* Displays:

  * 🌡️ Temperature
  * 🌤️ Weather condition
  * 💧 Humidity
* Simple and clean UI


🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API


🔑 Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Open the project folder:

```bash
cd your-repo-name
```

3. Get a free API key:

* Go to: https://openweathermap.org/api
* Sign up and generate an API key

4. Add your API key in `index.html`:

```js
const apiKey = 'YOUR_API_KEY';
```

5. Run the project:

* Open `index.html` in your browser
  OR
* Use GitHub Pages
